# cloudprojekt

## Container Image bilden

    git clone https://github.com/cntmodul/cloudprojekt
    cd cloudprojekt
    docker build -t cntmodul/cloudprojekt .
    docker push cntmodul/cloudprojekt
    
    
## Applikation starten

    git clone https://github.com/cntmodul/cloudprojekt
    kubectl apply -f cloudprojekt
